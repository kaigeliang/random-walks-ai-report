# Random Walks in AI: Interactive Supplement

This directory is ready for GitHub Pages. It contains a static HTML/CSS/JavaScript demo for the report `Random Walks: From Mathematical Foundations to Modern Applications in Artificial Intelligence`.

The page includes five animated examples:

- PageRank random surfer
- diffusion-model forward noising
- DeepWalk-style graph sampling
- random-walk Metropolis sampling
- epsilon-greedy reinforcement-learning exploration

To publish it, push the repository to GitHub, open **Settings > Pages**, and set the source to the `docs/` folder.
